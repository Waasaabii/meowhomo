## 设计规范

- **风格**: 全部二次元喵娘/猫元素，线性描边 2px，圆润可爱
- **尺寸**: 图标 24x24，头像 48x48，插画 256x256，Logo 128x128
- **颜色**: `currentColor` + CSS 变量，深色/浅色主题适配
- **动效**: CSS `@keyframes` 优先，复杂动画用 SMIL
- **路径**: `apps/web/src/assets/svg/`
