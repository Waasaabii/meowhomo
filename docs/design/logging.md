# MeowHomo 猫娘主题日志系统

MeowHomo 的日志系统是项目最大特色之一：所有模块的日志以 **8 只猫娘角色** 的身份输出，让枯燥的系统运维变得生动有趣。

## 1. 猫娘角色定义

| 角色 | 模块 | 性格设定 | 日志场景举例 |
|------|------|---------|------------|
| 🎀 **订阅喵** | 订阅管理 | 活泼好动，抱着鱼 | `[订阅喵] 成功拉取 CloudFlare 订阅，新增 12 个节点喵~` |
| 🎯 **策略喵** | 策略引擎 | 严肃巡逻，戴帽子 | `[策略喵] speed-first 组测速完成，最优节点: 东京-01 (45ms)` |
| 🔌 **端口喵** | 端口管理 | 慌张手忙脚乱 | `[端口喵] 端口 10001 被占用了喵！已自动迁移到 10005` |
| 🔧 **内核喵** | mihomo 引擎 | 戴眼镜修理工 | `[内核喵] mihomo v1.18.1 启动成功，加载 3 个策略组` |
| 📢 **通知喵** | 系统通知 | 拿喇叭大嗓门 | `[通知喵] TLS 证书将在 7 天后过期，请注意续签喵！` |
| 🔐 **TLS喵** | 证书/加密 | 优雅持钥匙 | `[TLS喵] 已通过 Let's Encrypt 签发新证书，有效期 90 天` |
| 📦 **备份喵** | 数据备份 | 默默搬运箱子 | `[备份喵] 定时备份完成，已保存至 backups/2026-03-01.zip` |
| 🏃 **测速喵** | 节点测速 | 运动装冲刺 | `[测速喵] 全量测速完成！平均延迟 89ms，最快: 香港-03 (23ms)` |

## 2. mihomo 原生日志拦截

mihomo 内核自身也会产生大量日志（连接建立、规则匹配、DNS 解析等）。处理流程：

```text
mihomo 内部日志
     ↓ Go 拦截 (替换 logger)
     ↓ 转换为 LogEntry protobuf
     ↓ gRPC stream 推送
     ↓ Rust 后端接收
     ↓ 分类到对应猫娘角色
     ↓ SSE 推送至前端
     ↓ Web 日志面板实时渲染
```

**分类规则**：
- 包含 `proxy` / `selector` / `fallback` 关键词 → 策略喵
- 包含 `listener` / `port` / `bind` → 端口喵
- 包含 `tls` / `cert` / `handshake` → TLS喵
- 其他 → 内核喵（默认）

## 3. 前端日志面板

### 3.1 完整日志页

- **实时滚动**：新日志自动追加到底部，支持锁定/解锁自动滚动。
- **过滤功能**：
  - 按角色过滤：点击猫娘头像 toggle 显示/隐藏该角色的日志。
  - 按级别过滤：info / warn / error 三级切换。
  - 关键词搜索：输入框实时过滤匹配。
- **操作按钮**：暂停（冻结当前日志流）、清空、下载（导出为 `.log` 文件）。

### 3.2 仪表板迷你日志

- 仪表板底部嵌入一个 mini 日志预览窗口（最近 20 条）。
- 只显示 warn + error 级别。
- 点击可跳转到完整日志页。

## 4. 日志存储

- 内存中保留最近 **2000 条**日志用于前端实时查看。
- 持久化存储到 SQLite（可配置保留天数，默认 7 天）。
- 支持按时间范围导出历史日志。
